Current & Past Projects


Bx Research Tool

The goal of this project is to facilitate behavioral economics / substance use research. I am developing it in Python with Django. This project involves dual deployment of MEAN and Django projects on the same server using Nginx.

Usage Scenario:
- Researcher wants to measure motivation for a commodity (e.g., alcohol)
- Researcher gives participants a link with embedded email and participant id
[Example: http://philspelman.com/research/task/researcher@someuniversity.edu/0001]
- Participant clicks the link and completes the task
- Raw and analyzed data are emailed back to the researcher (disabled in demo version)

Read about the project on my ResearchGate profile, or click the button to try it out.
